# **App Name**: Growth Tracker

## Core Features:

- Alarm Setting: Set alarms for specific times to trigger activity prompts.
- Note Input: Enter a short note describing the activity performed when the alarm rings.
- Keyword Extraction Tool: Extract keywords ('english', 'reading', 'work out', etc.) from the user's note using AI to categorize the activity.
- Time Tracking: Automatically track the duration of the activity (from alarm trigger to note entry).
- Google Sheets Integration: Seamlessly push extracted keywords and time spent data to a linked Google Sheet.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to inspire focus and self-improvement.
- Background color: Very light gray (#F0F0F0), almost white, to allow clear display of the main elements in the application.
- Accent color: Turquoise (#40E0D0) to highlight key interactive elements.
- Body and headline font: 'PT Sans' for readability and a touch of warmth.
- Use clean, minimalist icons to represent different activity categories.
- A simple and intuitive layout focused on ease of use for quick note entry.
- Subtle animations to acknowledge user input and progress.
